Welcome!

jLine 是一个用于创建横向的类似地铁站点导航的 jQuery 工具，未来会支持对纵向导航的支持。

jLine 可以用在展示步骤或流程的界面中增加用户体验。

usage

1、添加样式表到页面的头部
<link rel="stylesheet" href="content/j-line.css" />
2、添加 j-line 插件

要求 jQuery，推荐使用最新版本

<script type="text/javascript" src="scripts/jquery-1.7.2.js"></script>
<script type="text/javascript" src="scripts/j-line.js"></script>
3、例程：

<script type="text/javascript">

    // $("selector").jLine( options, [data] );

    $( function ()
    {
        var data_1 = ["v1.0", "v1.2", "v2.0", "v2.5"];
        var d = $( ".date" ).jLine( { contentPos: "top", connectLineWidth: "60px" }, data_1 );
        d.changeColor( 3, "orange" );

        var data_2 = ["开始", "卸载旧版本", "选择位置", "安装文件", "完成"];
        var setup = $( ".setup" ).jLine( { connectLineWidth: "70px" }, data_2 );
        setup.changeColor( 2, "blue", "forward" );

        var data_3 = ["福田口岸", "福民", "会展中心", "市民中心", "少年宫", "莲花北", "上梅林", "民乐", "白石龙", "深圳北", "红山", "上塘", "龙胜", "龙华", "清湖"];
        var station = $( ".station" ).jLine( { contentPos: "alternately", connectLineWidth: "50px" }, data_3 );
        station.changeColor( 4, "red", "forward" );
    } );
</script>

Document

options

optiontypedescriptiondefault value
showContent boolean 显示附加文本

文本中使用 => 符号可以将文本转为超链接，例如：

"sample=>http://www.sample.com"
true
contentPos string 文本位置，可选择的值：
  • top: 文字在[点]上方
  • bottom: 文字在[点]下方
  • alternately: 上下交替
 "bottom"
color string [点]的颜色
  • white - 白
  • gray - 灰
  • green - 绿
  • red - 红
  • blue - 蓝
  • purple - 紫
  • cyan - 青
  • yellow - 黄
  • orange - 橙
 "gray"
connectLineColor string 连接线的颜色，参考 color "gray"
connectLineWidth string/number 连接线长度 "30px"
stressEnds boolean 两端的 [点] 更突出（点稍微大一点点） true
dotClick function (callback)

点击指定的 [点] 产生的回调事件，发送参数：

function(event,dotIndex,dotObject)
  • event: 事件对象
  • dotIndex: 点击的点的索引
  • dotObject: [点] 对象
 $.noop

data

必须是数组

参考 option -> showContent 参数，例如：

[ "title1=>url", "title2", ... ]
